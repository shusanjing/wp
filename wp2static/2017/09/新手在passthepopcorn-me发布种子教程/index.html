passthepopcorn.me发布种子快速教程

2017年9月6日 392点热度
passthepopcorn.me以下简称PTP。
Power User (PU)等级需要至少发布一个种子（并且会员至少4周，上传至少80 GiB，比例高于1.05）：解锁IRC权限、长期账号无活动不会被删号（永久保号）。【2017年9月8日规则】
发布的信息填写注意点：
  • 需有MediaInfo信息。
  • 需有三张png原生截图，并且是不同的场景，截图放在mediainfo信息下面。官方图床ptpimg.me申请复杂，可直接使用picload.org。

流程：

    1. 制作种子(本篇略过)。
    2. 截图并上传到图床(本篇略过)。
    3. 在PTP发布种子：
      1. 填写imdb link自动填充基本信息。
      2. 添加海报图片链接。
      3. 填写影片编码、分辨率等信息。
      4. 填写影片发布说明正文：
        1. 使用potplayer打开视频-【属性】-【文件信息】，获取MediaInfo文件信息。
        2. 正文格式：

[/code]

      1. 信息预览后没问题即可提交：
    1. 提交后，到自己的发布页面查看发布的影片，并下载种子做种。
      1. Warning：Your torrent has been uploaded however, you must download your torrent from here because you didn't set the private flag or the announce URL when creating the torrent. Make sure to seed the new torrent instead of the one that you originally uploaded.
      2. 警告：您的torrent已经上传，但是，您必须从这里下载您的torrent，因为您创建洪流时没有设置私有标志或宣布URL。 确保种子新的洪流，而不是你最初上传的。
      1. 成功做种：
  2. 发布后面前标志默认是红色X，人工审核通过后会变成绿色√，一般需要一天时间。
【完】
