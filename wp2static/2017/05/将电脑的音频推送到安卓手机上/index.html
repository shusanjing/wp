将电脑的音频推送到安卓手机上

2017年5月30日 408点热度

概述

偶尔夜里瘫在床上用显示器看视频，声音小失去听觉体验严重影响整体欣赏，声音大又扰民，就有了无线耳机的需求，选购了罗技无线2.4g耳机，使用下来，该耳机体积大，使用闷热并不舒服，不用时候还要插着充电，否则下次拿出来使用很可能电量不足，并且每次使用需要手动切换声卡，用完再手动切换回原来声卡，简直蛋疼，一句话：用起来甚为麻烦。遂某次搬家直接丢弃。

今天使用 XRemoteServer 让安卓手机当做电脑鼠标键盘时，突然灵感一动：也许有让安卓手机充当电脑声卡的app？
就寻找到了Wireless speaker和SoundWire两款将电脑声音推送到手机的解决方案。

特点：
手机和电脑处于同一局域网即可使用。
PC和手机可以同时发声。
PC静音不影响手机发声。
优点：无需切换声卡、手机随时有电可以随想随用、可以使用自己喜欢的耳机耳塞。
缺点：将默认声卡的音频推送到手机上，有2s内的延迟。此问题一般专业播放器都有调整音频同步的功能可以解决，比如potplayer，快捷键[shit + ,]让音频提前 [shift + .]让音频滞后 [shift + /]让音频恢复默认。

使用总结：成本低，随想随用，不限耳机，对于只是临时想要电脑使用无线耳机的人，这是个可选的解决方案。缺点只有关键的一个：要忍受其延迟（基本都在0.5s-1.5s）。虽然播放器可以调整音频延迟的同步，但毕竟算不得方便。能逼死完美党：反复调整总觉得有点音画差异。

Wireless speaker

缺点：耳眼能明显感知延时。
优点：延时比SoundWire低一点（也许是错觉，未严格测试），并且服务端绿色免安装。

  • Wireless speaker Server （PC服务端）http://wifiaudio.boards.net/
  • wireless speaker for android（安卓客户端） https://play.google.com/store/apps/details?id=com.vnd.wifi_audio

SoundWire

缺点：耳眼能明显感知延时，并且无法正常退出，只能kill进程。
优点：无（客户端设置选项很多也许能算一个？没进一步研究这些功能）

  • SoundWireServer（PC服务端）https://georgielabs.net/
  • SoundWire (free) for android （安卓客户端） https://play.google.com/store/apps/details?id=com.georgie.SoundWireFree
