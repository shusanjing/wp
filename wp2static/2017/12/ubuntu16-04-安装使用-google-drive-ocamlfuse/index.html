ubuntu16.04 安装使用 google-drive-ocamlfuse

2017年12月16日 389点热度
1
2
3
测试服务商:itldc
root@ubuntu:~# uname -r
4.4.0-104-generic

使用PPA安装方式即可,
注:ubuntu14.04以后使用PPA需要先安装 software-properties-common 依赖包.

安装

PPA repository 安装方式:
1
2
3
4
sudo apt-get install software-properties-common //解决add-apt-repository: command not found问题
sudo add-apt-repository ppa:alessandro-strada/ppa
sudo apt-get update //必须更新
sudo apt-get install google-drive-ocamlfuse

授权

google-drive-ocamlfuse -headless -label me -id ##yourClientID##.apps.googleusercontent.com -secret ###yoursecret#####
